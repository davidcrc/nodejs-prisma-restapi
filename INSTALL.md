# nodejs-prisma-restapi

## init

```sh
$ npm init -y
```

## install prisma

```sh
$ npm install prisma typescript ts-node @types/node --save-dev
```

```sh
$ npx tsc --init
```

## init prisma - by default is postgresql

```sh
$ npx prisma init
```

## Run migrate with name init ( askf for name without --name )

```sh
$ npx prisma migrate dev --name init
```
